# Alpha Assistant Install Module

> Le module d'installation interactif de votre Alpha Assistant.

| Champ | Valeur |
|-------|--------|
| Module | `alpha-assistant-install` |
| Version | v1 |
| Phase | Foundation |
| Temps d'installation | 10-15 minutes |
| Coût | Gratuit |

---

## Ce que fait ce module

- **Interview interactive** : Claude vous pose 8 questions essentielles pour bien vous connaître
- **3 fichiers personnalisés générés automatiquement** :
  - `CLAUDE.md` (l'âme de votre assistant)
  - `context/CONTEXT.md` (votre contexte personnel)
  - `context/HISTORY.md` (votre journal de bord)
- **Configuration adaptée à votre profil** : étudiant, employé, entrepreneur, indépendant, ou mix
- **Mise en place du système d'auto-update** : Claude détectera automatiquement les changements importants à venir

---

## Comment l'installer

Depuis votre dossier Alpha Assistant ouvert dans Claude Code, lancez simplement :

```
/install module-installs/alpha-assistant-install
```

Claude Code lira ce module et démarrera l'interview interactive.

---

## Ce dont vous avez besoin avant de commencer

- 10 à 15 minutes devant vous, sans distraction
- Une idée claire de qui vous êtes et de ce que vous faites
- Une connexion Internet (Claude Code en a besoin)

---

## Ce qui se passe après l'installation

Une fois l'installation terminée, votre Alpha Assistant est immédiatement opérationnel.

À chaque future session, lancez simplement `/prime` au début pour que Claude charge votre contexte complet.
