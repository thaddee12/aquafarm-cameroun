# Clients

Livrables produits pour les clients d'INNOVA ALPHA.

Chaque client a son propre sous-dossier :
`[nom-client]/[type-livrable]_[version]_[AAAA-MM]`

Exemples : `aquafarm/`, `general-engineering/`

Ce qui va ici : chartes graphiques, plaquettes, documents administratifs, sites web livrés au client, contenus réseaux sociaux.
