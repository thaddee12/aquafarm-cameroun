# Sites Web

Sites internet produits pour les clients et projets d'INNOVA ALPHA.

Chaque projet a son propre sous-dossier nommé selon la convention :
`[client]_site-web_[version]_[AAAA-MM]/`

Exemples : `aquafarm_site-web_v1_2026-06/`, `innova-alpha_site-web_v1_2026-07/`

Ce qui va ici : code source, fichiers de configuration, exports, assets du site.
